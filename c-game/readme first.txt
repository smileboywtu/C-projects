			introduction



1）本程序基于推箱子改编而来，其中核心代码参考网上推箱子游戏
2）本程序全部采用纯C语言（由于是大一，其他GDI的API不太懂），运用GRAPHICS.H头文件，如果VC中没有安装easyx，请先解压压缩包进行安装（其实亦可以直接把EASYX中的graphic.h手动移植）
3）本程序的思路将由一张图片来展示（在根目录下）



			 OPERATION
1）可以使用方向键，也可以使用“W,S,A,D”,游戏中重新玩此局按“R”，按“ENTER”进行选择，按“ESC”退出；
